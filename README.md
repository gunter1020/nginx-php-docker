# nginx-php-docker

## Version info

- alpine `3.12`
- nginx `1.18.0`
- php `7.4.14`

## Links

- [GitHub - codecasts/php-alpine](https://github.com/codecasts/php-alpine)

## Quick Start

### Pull image

  ```bash
  docker pull gunter1020/nginx-php:latest
  ```

### Running

  ```bash
  docker run -it --rm -p 8080:80 gunter1020/nginx-php
  ```
